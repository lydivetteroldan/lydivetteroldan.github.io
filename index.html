Front End Developer

I like making websites and turning design into reusable code.

I am most comfortable when I am working with HTML, CSS, Sass, JavaScript, Git, Atom & Sketch.

Besides using my preferred tools, I develop web applications with a basic knowledge of Tachyons, Node, Express, MongoDB, Mongoose, Ruby on Rails, PostgreSQL, Grunt & Webpack.

After completing a bachelor’s degree in Computer Information Systems, I taught myself web development using HTML, CSS and basic JavaScript with the help of books, tutorials and online coding courses. Recently, I learned how to build full stack web applications while being part of General Assembly’s Web Development Immersive program.

I have a multi-disciplinary work experience in developing and editing creative content, proofreading, translation and customer service. I have also worked as a Front-End Developer in a software development startup, and as a freelancer – these jobs have lead me to build user interfaces for government agencies, marketing workshops and entertainment companies.
