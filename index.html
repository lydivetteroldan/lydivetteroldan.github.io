About

I have a multi-disciplinary work experience in developing and editing creative content, proofreading, translation and customer service. I have also worked as a Front-End Developer in a software development startup, and as a freelancer, building user interfaces for government, marketing and entertainment agencies.

After completing a bachelor’s degree in Computer Information Systems, I taught myself web development using HTML, CSS and basic JavaScript with the help of books and online coding courses. Recently, I learned how to build full stack web applications as part of General Assembly’s Web Development Immersive program in Boston.

I am confident and happy working with HTML, CSS, Sass, JavaScript, Git, Atom & Sketch.

Besides using my preferred tools, I develop web applications with a basic knowledge of Tachyons, Node, Express, MongoDB, Mongoose, Ruby on Rails, PostgreSQL, Grunt & Webpack.
